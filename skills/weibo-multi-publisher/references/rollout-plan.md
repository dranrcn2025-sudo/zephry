# 微博多账号发布 Skill Rollout Plan

## 第一阶段：骨架
- 建 skill 目录
- 定义配置格式
- 定义账号映射
- 定义素材输入方式
- 定义 dry-run 与正式发布模式

## 第二阶段：登录态接入
- 收 3 个微博账号 storage state
- 校验每个账号是否仍然在线
- 验证是否能打开微博发布页

## 第三阶段：发布流程打通
- 纯文案发布
- 图文发布
- 单视频发布
- 逐账号结果回报

## 第四阶段：稳定性
- 失败重试
- 风控提示
- 分账号错峰发布
- 变体文案支持（如用户需要）

## 建议输入

### 文案文件
纯文本 UTF-8。

### 媒体文件
支持本地路径列表，第一版不做远程下载。

### 返回结果
建议固定结构：
- account
- mode
- status
- detail
- post_url（若拿得到）

## 风险
- 微博页面结构变化快
- 多账号同内容可能触发风控
- 视频上传耗时长
- 登录态可能失效

## 第一版成功标准
- 至少能对 1 个账号完成 dry-run
- 至少能对 1 个账号完成真实发布验证
- 其余账号可复用同一流程接入
