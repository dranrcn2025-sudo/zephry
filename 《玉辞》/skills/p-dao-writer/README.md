# p-dao-writer Skill

Priest（p大）风格小说写作技能包。

## 概述

这个skill基于Priest（p大）的小说创作风格，专门用于生成和优化具有p大特色的小说文本。p大是中国知名网络文学作家，代表作包括《六爻》、《杀破狼》、《镇魂》、《默读》等，以其文笔老练、人物立体、情感深刻著称。

## 核心特点

### 1. 文风特色
- **文白夹杂的典雅**：现代白话中自然融入文言词汇，典雅而不晦涩
- **冷幽默的智慧**：在严肃叙事中穿插巧妙幽默，犀利而不低俗
- **情感的克制**：情感表达含蓄内敛，多一分则滥情，少一分则冷漠
- **心理的深度**：深入挖掘人物内心矛盾，展现人性复杂面
- **人物视角的坚持**：**这是p大风格的核心秘密**。始终通过人物的眼睛看世界，通过人物的心思想问题。不是作者向读者解释逻辑，而是展现人物的思考过程。读者不是被“告知”，而是“跟随”人物去发现。

### 2. 人物塑造
- **拒绝脸谱化**：好人也有阴暗面，坏人也有可怜处
- **成长轨迹清晰**：人物有明确的成长过程和转变节点
- **关系网络复杂**：通过人物关系展现不同侧面

### 3. 叙事技巧
- **细节真实感**：即使是仙侠玄幻，细节也要真实可信
- **伏笔巧妙**：伏笔隐蔽，呼应自然，让读者有“原来如此”的惊喜
- **节奏张弛**：紧张情节与舒缓描写交替，给读者喘息空间

## 使用方法

### 1. 激活技能
当需要创作或模仿p大风格的小说时，系统会自动加载此技能。也可以手动触发：
- “用p大的风格写一个修真故事”
- “模仿《六爻》的风格写一段”
- “帮我写个仙侠文，要p大那种感觉”

### 2. 写作流程
1. **理解需求**：明确故事类型、人物设定、核心情节
2. **风格匹配**：根据p大风格特点进行创作
3. **五层自检**：完成硬性规则、风格一致性、内容质量、文学性、整体风格五层检查
4. **修改优化**：根据质检报告进行针对性修改

### 3. 关键检查点
- **文白比例**：文言占比10-20%，自然流畅
- **情感克制**：避免过度煽情，用动作、细节间接表达
- **人物复杂**：确保主要人物有内心矛盾和成长轨迹
- **细节真实**：即使是虚构世界，细节也要逻辑自洽
- **幽默自然**：冷幽默要与情节、人物相关，不刻意

## 五层自检体系

### L1 硬性规则检查
- 禁用滥情词汇（泪水模糊、心如刀绞等）
- 检查文白自然度
- 防止人物脸谱化
- 确保细节真实
- **检查人物视角**：避免作者解释句式（特别是“不是……而是……”）

### L2 风格一致性检查
- 开头是否从具体场景切入
- 节奏是否张弛有度
- 文笔是否符合p大特色
- 冷幽默是否自然巧妙

### L3 内容质量检查
- 人物是否有深度和成长
- 主题是否深刻且自然流露
- 情节逻辑是否自洽
- 情感分寸是否恰当

### L4 文学性检查
- 文笔是否具有美感
- 是否有象征隐喻
- 节奏是否优美
- 是否留有余味

### L5 整体风格终审
- 是否有p大风格辨识度
- 情感是否有温度
- 是否有智慧闪光
- 各要素是否和谐统一

## 示例对比

### AI常见问题 vs p大风格修正

**问题原文（AI常见）：**
> 程潜心里很难过，他恨父母为什么要抛弃他，眼泪忍不住流了下来。他对着天空大喊：“为什么！为什么要这样对我！”他感到无比的痛苦和绝望。

**p大风格修正：**
> 程潜没哭，只是看着爹娘模糊的背影。心里那点凉薄的想法又冒了出来：卖了就卖了吧，反正他们眼里从来就没有他。他抹了把脸，转身跟上师父，再没回头。

**修改要点：**
1. 情感从直白改为克制（“没哭”代替“眼泪流了下来”）
2. 行为从夸张改为平淡（“抹了把脸”代替“对着天空大喊”）
3. 心理从简单改为复杂（“凉薄的想法”代替“痛苦绝望”）
4. 结尾从宣泄改为含蓄（“再没回头”留有余味）

## 适用场景

1. **仙侠修真小说**：如《六爻》风格的修真故事
2. **古代传奇小说**：如《杀破狼》风格的权谋战争
3. **现代悬疑小说**：如《默读》风格的刑侦破案
4. **奇幻玄幻小说**：如《镇魂》风格的灵异奇幻
5. **任何需要深刻人物和典雅文风的小说创作**

## 注意事项

1. **避免过度模仿**：模仿的是精神内核，不是表面形式
2. **保持自然流畅**：文白夹杂要自然，不生硬
3. **把握情感分寸**：克制不等于冷漠，要有温度
4. **注重逻辑自洽**：即使是玄幻，也要内部逻辑一致
5. **尊重原创精神**：在模仿基础上要有自己的思考和创造

## 学习建议

1. **多读原作**：深入阅读p大的作品，体会其风格精髓
2. **分析练习**：分析经典片段，理解其创作技巧
3. **模仿写作**：从模仿开始，逐渐形成自己的理解
4. **反复修改**：利用五层自检体系不断优化作品
5. **寻求反馈**：让他人阅读，获取客观评价
