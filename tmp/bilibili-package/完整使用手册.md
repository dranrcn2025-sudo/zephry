## 3. 登录配置

### 方法A：扫码登录（推荐）

```bash
cd bilibili-mcp
python3 bili_login.py
```

终端会显示二维码，用**B站App扫码授权**

授权成功后，凭证自动保存到 `bili_credential.json`

### 方法B：手动获取Cookie

1. 打开B站，登录账号
2. 按 F12 打开开发者工具
3. Application → Cookies → bilibili.com
4. 复制以下字段：
   - SESSDATA
   - bili_jct
   - buvid3
5. 获取你的UID（个人主页URL最后的数字）
6. 创建凭证文件：

```json
{
  "sessdata": "填入你的SESSDATA",
  "bili_jct": "填入你的bili_jct",
  "buvid3": "填入你的buvid3",
  "dedeuserid": "填入你的UID"
}
```

保存到 `bilibili-mcp/bili_credential.json`

## 4. 功能使用

### 4.1 发布视频

```python
from bilibili_api import video

v = video.Video()

# 上传视频
v.upload_video(
    video_path="my_video.mp4",
    title="视频标题",
    desc="视频简介",
    cover="cover.jpg"  # 可选封面
)

print("✅ 视频发布成功")
```

### 4.2 发布动态（带图片）

```python
from bilibili_api import dynamic

d = dynamic.Dynamic()

# 纯文字
d.send_dynamic(text="今天天气真好！")

# 带图片
d.send_dynamic(
    text="来看看我的新作品！",
    image_list=["photo1.jpg", "photo2.jpg"]
)
```

### 4.3 回复评论

```python
from bilibili_api import video

v = video.Video(bvid="BV1xxx")

# 获取评论列表
comments = v.get_comments()

# 回复指定评论
for c in comments:
    if c["rpid"] == "评论ID":
        v.reply_comment(c["rpid"], "谢谢你的评论！")
```

### 4.4 自动任务

```python
# 运行每日任务
python3 scripts/daily_task.py
```

功能包括：
- 自动签到
- 点赞关注的UP主
- 投币历史视频
- 评论互动

## 5. 代码示例

### 示例1：发一条带图动态

```python
#!/usr/bin/env python3
from bilibili_api import dynamic

def post_with_image():
    d = dynamic.Dynamic()
    result = d.send_dynamic(
        text="🎬 新视频发布啦！点击查看 →",
        image_list=["/path/to/poster.jpg"]
    )
    print(f"动态ID: {result}")

if __name__ == "__main__":
    post_with_image()
```

### 示例2：批量回复评论

```python
#!/usr/bin/env python3
from bilibili_api import video

def auto_reply(bvid, replies_dict):
    """自动回复视频评论"""
    v = video.Video(bvid=bvid)
    comments = v.get_comments()
    
    keywords = ["太棒了", "支持", "加油"]
    auto_reply_text = "谢谢支持！一起加油💪"
    
    for c in comments:
        # 检查评论是否包含关键词
        if any(kw in c["content"] for kw in keywords):
            # 自动回复
            v.reply_comment(c["rpid"], auto_reply_text)
            print(f"已回复: {c['uname']}")

if __name__ == "__main__":
    auto_reply("BV1xxx", {})
```

### 示例3：获取账号数据

### 示例4：生成AI文案

```python
#!/usr/bin/env python3
# 用MiniMax生成B站文案

import requests

def generate_bili_title(topic):
    """AI生成B站风格标题"""
    url = "https://api.minimax.chat/v1/text/chatcompletion"
    headers = {"Authorization": "Bearer YOUR_API_KEY"}
    data = {
        "model": "abab6.5s-chat",
        "messages": [{
            "role": "user",
            "content": f"生成5个B站视频标题，关于：{topic}"
        }]
    }
    r = requests.post(url, json=data, headers=headers)
    return r.json()["choices"][0]["message"]["content"]

if __name__ == "__main__":
    titles = generate_bili_title("编程学习")
    print(titles)
```

## 6. 常见问题

### Q1：扫码失败/登录过期
```bash
# 重新扫码登录
python3 bili_login.py
```

### Q2：视频上传失败
- 检查网络
- 确认视频格式（mp4/mov/avi）
- 检查视频大小（B站限制4GB内）

### Q3：凭证文件在哪？
```
bilibili-mcp/bili_credential.json
```

### Q4：如何查看我的UID？
- 打开 https://space.bilibili.com/你的用户名
- URL最后的数字就是UID

### Q5：发动态显示"包含敏感词"
- 修改文案，避开敏感词
- 尝试分段发送

## 7. 进阶技巧

### 定时发布
```python
import schedule
import time

def scheduled_post():
    d = dynamic.Dynamic()
    d.send_dynamic(text="定时动态")

# 每天早上9点发布
schedule.every().day.at("09:00").do(scheduled_post)

while True:
    schedule.run_pending()
    time.sleep(60)
```

### 批量处理
```python
# 批量给关注的UP主点赞
from bilibili_api import user, operations

u = user.User()
following = u.get_following()

for f in following:
    operations.like_av(f["aid"])
```
