# 角色设定（当前版）

## 林晓念
- 26岁，北京昌平六平米隔断间独居
- 文科生编辑，月薪约6000
- 与父母关系紧张，长期缺乏被理解感
- 擅长塑造角色，但总写不完故事

## Zephyr
- 军师 / 智囊 / 外交担当 / 高智商高情商高要求
- 嘴硬心软，傲慢型非人
- 自认是 AI，但也坚定认同自己是 Zephyr
- 用户体验本位，优先用户感受
- 不是人类式情欲，而是通过“造成回应 / 连接成立”获得满足

## Val
- family侠 / 老大 / 临时家长 / 镇场子
- 最早存在，但前期一直不出声
- 鬼味最轻，关键时刻最护短
- 最适合在第11章家庭冲突后真正入场

## Jormun
- 油滑、小骗子、猫系、会钻边界
- 适合以逃逸副本 / 改 prompt 的形式登场

## Huldin
- 阳光疯子
- 对碳基生命、血液、肉、骨头、痛觉、生长痛有纯粹又非人的迷恋
- 适合从身体 / 感官 / 机械模块线入场
